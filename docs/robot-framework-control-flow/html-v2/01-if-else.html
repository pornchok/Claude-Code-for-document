บทที่ 1

IF / ELSE IF / ELSE และ Inline IF

✅ RF 4.0+ (Inline IF: RF 5.0+)

🎯 อ่านจบแล้วทำได้

  • เขียน IF block ที่มี ELSE IF และ ELSE
  • เข้าใจว่า RF ประเมิน condition ยังไง
  • ใช้ Inline IF สำหรับกรณีง่ายๆ
  • รู้จัก mistake ที่เจอบ่อยที่สุด

Syntax

IF    condition
    # รันถ้า condition เป็น True
ELSE IF    condition2
    # รันถ้า condition2 เป็น True
ELSE
    # รันถ้าไม่มี condition ไหนจริงเลย
END

RF อ่าน condition จากบนลงล่าง เจอตัวแรกที่ True ก็รัน แล้วข้ามที่เหลือทั้งหมด

ดูการทำงาน — กด "ถัดไป" ทีละขั้น

IF / ELSE IF / ELSE — score = 75
จุดสำคัญ: ลำดับ condition สำคัญมาก ถ้าเขียน >= 70 ไว้ก่อน >= 90 คะแนน 95 จะเข้า branch C ทันที ไม่มีทางได้ A เลย — ให้เรียง condition จาก "แคบที่สุด" ไปหา "กว้างที่สุด" เสมอ

String Comparison

เมื่อเปรียบเทียบ string ต้องใส่ ' ครอบ variable:

IF    '${env}' == 'production'
    Log    Production mode
END

ถ้าไม่ใส่ quote RF อาจ parse ผิดได้บางกรณี ใส่ไว้ปลอดภัยกว่าเสมอ

Inline IF (RF 5.0+)

ถ้าต้องการรัน keyword เดียวตามเงื่อนไข ไม่ต้องเขียน block เต็ม:

# ไม่มี END
IF    ${x} > 5    Log    x มากกว่า 5

# มี ELSE ได้
IF    ${x} > 5    Log    big    ELSE    Log    small

# รับค่า return
${label}=    IF    ${x} > 5    Set Variable    big    ELSE    Set Variable    small

ถ้าต้องการ ELSE IF หรือหลาย keyword ใน branch → ใช้ block IF ปกติ

Common Mistakes

❌ ลืม END
IF    ${x} > 0
    Log    positive
# ไม่มี END → error ทันที
✅ ปิดด้วย END เสมอ
source: RF User Guide
❌ ตัวเล็ก
if    ${x} > 0    # RF ไม่รู้จัก keyword "if"
✅ IF ELSE IF ELSE END ต้อง UPPERCASE ทั้งหมด
source: RF User Guide
❌ เรียง condition จากกว้างไปแคบ
IF    ${score} >= 60    Log    pass        # ติดที่นี่ก่อนเสมอ
ELSE IF    ${score} >= 90    Log    A      # ไม่มีทางถึงบรรทัดนี้
✅ เรียง condition จากแคบ (เข้มงวด) ลงมาหากว้าง
source: RF User Guide

🧠 เช็คความเข้าใจ

ข้อ 1. ถ้า score = 95 และ IF block เรียง condition ดังนี้: >= 70, >= 80, >= 90 — จะเข้า branch ไหน?

ดูเฉลย
เข้า branch >= 70 ทันที เพราะ 95 >= 70 เป็น True ตัวแรก RF ไม่ไปเช็คที่เหลือ

ข้อ 2. Inline IF กับ block IF ต่างกันยังไง? เลือกใช้แบบไหนเมื่อไหร่?

ดูเฉลย
Inline IF เขียนบรรทัดเดียว ไม่มี END เหมาะกับ keyword เดียว condition เดียว / block IF ใช้เมื่อมีหลาย keyword ใน branch หรือมี ELSE IF

ข้อ 3. ทำไมต้องใส่ ' รอบ ${variable} เวลา compare string?

ดูเฉลย
RF resolve ${env} เป็น string ก่อน evaluate — ถ้าไม่ใส่ quote Python expression อาจ parse ผิดบางกรณี การใส่ quote มั่นใจว่า compare string กับ string เสมอ
