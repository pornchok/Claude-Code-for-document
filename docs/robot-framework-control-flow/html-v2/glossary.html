Control Flow Keywords

IF
เริ่ม conditional block — ต้องตามด้วย condition และปิดด้วย END
RF 4.0+ · RF User Guide
ELSE IF
Branch เพิ่มเติมใน IF block ตรวจสอบเฉพาะเมื่อ condition ก่อนหน้าเป็น False
RF 4.0+ · RF User Guide
ELSE
Branch สุดท้ายที่ทำงานเมื่อทุก condition ก่อนหน้าเป็น False
RF 4.0+ · RF User Guide
Inline IF
IF แบบบรรทัดเดียว ไม่ต้องมี END เหมาะกับ keyword เดียว / condition เดียว — รับค่า return ได้ด้วย
RF 5.0+ · RF User Guide
FOR
เริ่ม loop — ต้องระบุ loop variable, รูปแบบ (IN/IN RANGE/...) และปิดด้วย END
RF 4.0+ · RF User Guide
FOR IN
วน loop ผ่าน list ทีละ item — ใช้ @{} เพื่อ unpack list variable
RF 4.0+ · RF User Guide
FOR IN RANGE
วน loop ตามช่วงตัวเลข — IN RANGE [start=0] end [step=1] ค่า end ไม่ถูกรวม (เหมือน Python range())
RF 4.0+ · RF User Guide
FOR IN ENUMERATE
วน loop ได้ทั้ง index และ item — start=N เพื่อเริ่ม index จากเลขอื่น (default: 0)
RF 4.0+ (start=: RF 6.0+) · RF User Guide
FOR IN ZIP
วน loop ผ่าน 2+ list คู่ขนาน — ต้องเป็น list variable ไม่ใช่ literal, หยุดเมื่อ list ที่สั้นที่สุดหมด
RF 4.0+ · RF User Guide
WHILE
วน loop ตราบที่ condition เป็น True — เช็ค condition ก่อนทุก iteration (ถ้า False ตั้งแต่แรก loop body ไม่ทำงานเลย)
RF 5.0+ · RF User Guide
limit
Parameter ของ WHILE กำหนดจำนวนรอบสูงสุด (default: 10,000) — ถ้าเกิน FAIL ทันที รองรับ: limit=100, limit=10 seconds, limit=NONE
RF 5.0+ · RF User Guide
BREAK
หยุด loop ทันที ออกจาก loop ไปรัน keyword ถัดจาก END — ใช้ได้ทั้งใน FOR และ WHILE
RF 5.0+ · RF User Guide
CONTINUE
ข้าม iteration ปัจจุบัน วนกลับไปเช็ค condition ใหม่ — ใช้ได้ทั้งใน FOR และ WHILE
RF 5.0+ · RF User Guide
END
ปิด block ของ IF, FOR, WHILE — ขาดไม่ได้ (ยกเว้น Inline IF ที่ไม่มี END)
RF 4.0+ · RF User Guide

คำศัพท์ทั่วไป

Condition
Python expression ที่ประเมินผลเป็น True/False — ใช้ใน IF และ WHILE เช่น ${x} > 10, '${name}' == 'Alice'
iteration
การวนหนึ่งรอบของ loop เช่น FOR loop วน 5 รอบ = 5 iterations
infinite loop
Loop ที่ condition ไม่เคยเป็น False และไม่มี BREAK — ใน RF จะ FAIL เมื่อถึง limit (ไม่ค้างตลอดไป)
unpack
การแตก list ออกเป็น item ย่อยๆ — ทำด้วย @{} ใน FOR loop เช่น FOR ${x} IN @{myList}
