ตัวแปรที่ใช้บ่อย

รูปแบบใช้เมื่อตัวอย่าง
${variable}เก็บค่าทั่วไป (string, number, object)${score}= Set Variable 85
@{list}เก็บ list — ใช้ใน FOR loop เพื่อ unpack@{items}= Create List a b c
สำคัญ: FOR ${x} IN @{list} วนทีละ item แต่ FOR ${x} IN ${list} ได้ list ทั้งก้อนเป็น item เดียว

Built-in Keywords ที่ใช้บ่อยในเอกสารนี้

Keywordทำอะไร
Logprint ข้อความลง log
Set Variableกำหนดค่าให้ตัวแปร
Evaluateคำนวณ Python expression — ใช้บ่อยใน WHILE เพื่อ update ค่า เช่น ${i}= Evaluate ${i} + 1
Should Be Equalassert ว่าค่าสองอันเท่ากัน
Create Listสร้าง list ใหม่
Run Keyword And Return Statusรัน keyword แล้วคืน True/False แทนที่จะ fail ทันที
